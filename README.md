## 📋 Опис

FieldCalc - це веб-додаток для геодезичних розрахунків, призначений для визначення площ та периметрів земельних ділянок у військових та гуманітарних цілях.

## ✨ Основні функції

- 📐 Розрахунок площі та периметру полігонів
- 🧭 Автоматичне визначення магнітного схилення
- 🗺️ Підтримка UTM координатних систем (зони 34-38N для України)
- 📊 Експорт в різні формати (CSV, KML, GeoJSON, PDF, PNG)
- 🎯 Інтерактивна карта з маркерами точок


## ⚠️ Важливо

- Додаток призначений для **попередніх розрахунків**
- Для офіційних документів звертайтесь до ліцензованих геодезистів
- Всі розрахунки виконуються локально в браузері
- Координати не передаються на сервер
