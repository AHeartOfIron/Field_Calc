# FieldCalc

Професійний калькулятор для геодезичних зйомок з підтримкою UTM координат та візуалізацією на карті.

## Особливості

- 🗺️ Інтерактивна карта з підтримкою різних шарів
- 📊 Автоматичні розрахунки азимутів, відстаней та площ
- 🎯 Підтримка UTM координат (зони 1-60)
- 📁 Імпорт/експорт даних (KML, CSV, JSON)
- 🔧 Автокорекція похибки замикання
- 🌍 Пошук населених пунктів
- 📱 Адаптивний інтерфейс

## Використання

1. Оберіть UTM зону
2. Встановіть магнітне схилення
3. Введіть координати точок (LM, BM, SP, TP)
4. Натисніть "Розрахувати"
5. Переглядайте результати на карті

## Підтримувані формати

- **KML** - для Google Earth
- **CSV** - табличні дані
- **JSON/GeoJSON** - для ГІС систем

## Технології

- HTML5, CSS3, JavaScript
- Leaflet.js - інтерактивні карти
- Proj4.js - проекції координат
- Turf.js - геопросторові розрахунки
