# FieldCalc - Геодезичний калькулятор полігонів

🌐 **Тільки онлайн використання** | © 2025 Illia Usachov

## 📋 Опис

FieldCalc - це веб-додаток для геодезичних розрахунків, призначений для визначення площ та периметрів земельних ділянок у військових та гуманітарних цілях.

## ✨ Основні функції

- 📐 Розрахунок площі та периметру полігонів
- 🧭 Автоматичне визначення магнітного схилення
- 🗺️ Підтримка UTM координатних систем (зони 34-38N для України)
- 📊 Експорт в різні формати (CSV, KML, GeoJSON, PDF, PNG)
- 🎯 Інтерактивна карта з маркерами точок
- 📱 Адаптивний дизайн для мобільних пристроїв

## 🎯 Підтримувані формати експорту

### Для ГІС систем:
- **GeoJSON** - для ArcGIS Pro, QGIS, Google Earth
- **KML** - для Google Earth, ArcGIS
- **GeoPackage SQL** - для створення .gpkg в QGIS

### Для звітності:
- **PNG** - 4K карти з детальною інформацією
- **PDF** - професійні звіти з картами
- **CSV/TSV** - табличні дані
- **TXT** - текстові звіти

## 🚀 Використання

1. Відкрийте додаток у веб-браузері
2. Оберіть UTM зону (автоматично визначається по геолокації)
3. Введіть координати точок (LM, BM, SP, TP1, TP2...)
4. Натисніть "Розрахувати"
5. Експортуйте результати в потрібному форматі

## 🛠️ Технології

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Карти**: Leaflet.js з підтримкою різних базових карт
- **Проекції**: Proj4.js для конвертації координат
- **Експорт**: jsPDF, html2canvas, JSZip
- **Геометрія**: Turf.js для геопросторових розрахунків

## 📊 Підтримувані координатні системи

- **UTM зони**: 34N, 35N, 36N, 37N, 38N (для України)
- **Датум**: WGS84
- **Формати**: UTM метричні координати, Lat/Lng

## ⚠️ Важливо

- Додаток призначений для **попередніх розрахунків**
- Для офіційних документів звертайтесь до ліцензованих геодезистів
- Всі розрахунки виконуються локально в браузері
- Координати не передаються на сервер
